YOLO安装和使用

本教程在Linux(Debian11)/Windows上测试，主要是在Linux上运行YOLO，以及Windows下标注数据

YOLO的官网在这里访问： YOLO官网

环境配置

创建conda环境

首先，创建conda环境，使用python=3.9.12 


conda create -n rdd python=3.9.12 -y
conda activate rdd

安装Pytorch

在实验中，我们使用pytorch2.8，需要注意你的系统版本是否太老旧，没有办法使用这个pytorch的版本。 


pip install torch==2.8.0 torchvision==0.23.0 torchaudio==2.8.0 --index-url https://download.pytorch.org/whl/cu129

安装YOLO(Ultralytics)

我们使用最新的Ultralytics，因为我们认为他们是向下兼容的，但是我们会贴出这个教程使用的具体版本号。

如果您观看这个教程的时间线比较远(2026/04/08)，请先看后面的具体列出版本号，不要直接下载。 

pip install -U ultralytics
                

pip install ultralytics-opencv-headless

验证YOLO

下载YOLO模型

YOLO有已经预训练过的模型可以在官网下载: YOLO模型下载

可以根据自己的需求选择使用的模型，这里使用YOLO26-seg模型，下载地址: YOLO-seg模型下载

我们使用YOLO官网提供的图片： 


from ultralytics import YOLO
# 加载模型
model = YOLO("/mnt/data3/yolo26m-seg.pt")
# 运行检测
results = model("./bus.jpg")
# 展示结果
for result in results:
    # result.show()  # 如果你是桌面环境的话，这行可以解开注释
    result.save(filename="result.jpg")  # 保存
注意，你应该检查所有的路径，确保和你的模型/图片路径一致

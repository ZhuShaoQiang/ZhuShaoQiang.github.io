环境配置

创建conda环境

你可以直接在YOLO的环境中安装，那样的话，可以跳过这个小章节。

首先，创建conda环境，使用python=3.9.12 


conda create -n labelme python=3.9.12 -y
conda activate labelme

安装LabelMe


在这里下载应用程序:
LabelMe应用程序
                

pip install labelme

安装了之后，在conda环境中，直接执行命令，即可打开界面，界面如下图所示：

转换数据

现在我们得到了一个带有通用标注json文件的数据集，现在需要转化为YOLO可识别的模式（COCO数据集格式）。

我们选择的数据转换工具是： LabelMe2YoloV8-segment转换工具

下载了这个工具之后，这样运行(使用上一章节创建的conda环境即可)： 


python .\convert_folder.py E:\ttttt\imgs E:\ttttt\dataset 0.7

其中的path是指这个数据集所在的根目录，如果你需要移动数据集，请修改这个值。

微调模型

使用下面的代码进行微调训练： 


from ultralytics import YOLO

# 加载预训练模型
model = YOLO("yolo26m-seg.pt")
# 训练模型，当然，你不加载预训练模型也没事
results = model.train(data="你的yaml路径，记得修改yaml中的path", epochs=20, imgsz=640)

虽然代码中没有保存，但是运行微调时，会自动保存最后一个和最好的那一个。
